Atk: s
The time in seconds for the first phase (attack) where the gain rise from 0 to 1
dec: s
The time in seconds for the second phase (decay) where the gain drops from 1 to sustain level
rel: s
The time in seconds for the final phase where the gain drops from the sustain level to zero when you release the note
Sust:
The sustain level, should be a value between 0 and 1
